GLB Debug Viewer

Loads each exported character in isolation, frames it automatically, and reports suspicious export signals like missing skinned meshes, missing bones, invalid bounds, or missing animation clips.

How to use this page
  • If a model loads here but not in the simulator, the export is probably fine and the scene logic is the issue.
  • If a model has zero skinned meshes or zero bones, the Blender export likely dropped the armature.
  • If the height is absurdly tiny or huge, transforms were probably not applied before export.
  • If the model is present but clipped or disappears when orbiting, stale bounds or frustum culling may be involved.
